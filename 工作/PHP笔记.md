# 一、PHP简历

# 二、php基本语法
2. php文件格式
	+ 代码需要在尖括号里面：
		```
		<?php
			// 这是一个注释
			echo "hello world";
		?>
		```
	+ <?php ?>

2. 注释
	+ 单行注释：
		+ `// 这是一个注释`
		+ ` # 这是一个注释`
	+ 多行注释：`/* 这是一个注释 */`

3. 拼接字符串
	+ 可以使用`.`运算符来拼接字符串
	+ 例如：`$a = "hello"; $b = "world"; echo $a . $b;`

4. 单引号和双引号
	+ 单引号：`'hello world'`
		+ 不可以嵌入变量
	+ 双引号：`"hello world"`
		+ 可以嵌入变量
	```
		$str = "what the fuck";
		echo "output $str";
		echo "<br>";
		echo 'output $str';
	```

5. 调试，查看变量的值
	+ var_dump()函数：
		+ 可以查看变量的类型和值
		+ 没有返回值
		+ 例如：`var_dump($a);`
		+ 输出结果：`int(10)`

6. 变量
	+ 以`$`开头，跟着变量的名称，用=进行赋值，将右边的值赋给变量。
	+ 变量名只能包含字母、数字和下划线
	+ 变量名不能以数字开头

7. 变量作用域
	+ 我们在函数内访问全局变量，需要使用`global`关键字引入
	+ static关键字：
		+ 可以在函数内定义一个静态变量，静态变量的值在函数调用之间保持不变
	+ 例如：
	```
		//函数内使用全局变量
		function test(){
			global $a;
			echo $a;
		}
		$a = 10;
		test();

		//static关键字
		function test2(){
			static $a = 0;
			$a++;
			echo $a;
		}
		test2();	//1
		test2();	//2
		test2();	//3
	```

8. 常量
	+ 常量名要用大写字母
	+ 语法：
		`define("常量名", 值);    //已弃用`
		`const 常量名 = 值;`
	+ 例如：
		```
			define("PI", 3.1415926);	//已弃用
			echo PI;	//3.1415926

			const PI = 3.1415926;
			echo PI;	//3.1415926
		```

9. 变量
	+ 变量类型：整型、浮点型、字符串、布尔型、数组、对象、资源、空类型
	+ 类型检测：
		+ is_int()：检测变量是否为整型
		+ is_float()：检测变量是否为浮点型
		+ is_string()：检测变量是否为字符串
		+ is_bool()：检测变量是否为布尔型
		+ is_array()：检测变量是否为数组
		+ is_object()：检测变量是否为对象
		+ is_resource()：检测变量是否为资源
		+ is_null()：检测变量是否为空
	+ 类型转换：
		+ intval()：将变量转换为整型
		+ floatval()：将变量转换为浮点型
		+ strval()：将变量转换为字符串;NULL和布尔型会转换为空字符串
		+ boolval()：将变量转换为布尔型
		+ arrayval()：将变量转换为数组
		+ objectval()：将变量转换为对象
		+ resourceval()：将变量转换为资源
		+ nullval()：将变量转换为空

10. 浮点数
	+ 浮点型数据，是存在精度问题的
	+ 永远不要相信浮点数结果精确到了最后一位，也永远不要比较两个浮点数是否相等。

11. 字符串
	+ 双引号和单引号的区别：
		+ 双引号可以解析变量，单引号不能解析变量
		+ 双引号可以解析转义字符，单引号不能解析转义字符
	+ 字符串定义类型：单引号形式、双引号形式、heredoc形式
	+ 字符串转换
		+ strval()：将变量转换为字符串;NULL和布尔型会转换为空字符串
	+ 字符串长度：strlen()函数
		+ 例如：`echo strlen("hello");`
		+ 输出结果：5
	+ 字符串拼接：.运算符
		+ 例如：`echo "hello" . "world";`
		+ 输出结果：helloworld
	+ 字符串截取：substr()函数
		+ 例如：`echo substr("hello", 1, 3);`
		+ 输出结果：ell
	+ 字符串替换：str_replace()函数
		+ 例如：`echo str_replace("hello", "world", "hello world");`
		+ 输出结果：world world

12. 数组
	+ 数组类型：索引数组、关联数组
	+ 数组定义：
		+ 索引数组：`$arr = array(1, 2, 3);`
		+ 关联数组：`$arr = array("name" => "John", "age" => 30);`
	+ 数组操作：
		+ 添加元素：`$arr[] = 4;`
		+ 删除元素：`unset($arr[0]);`
		+ 修改元素：`$arr[0] = 100;`

13. NULL
	+ NULL是一个特殊变量类型
	+ NULL出现的情况
		+ 被赋值为NULL
		+ 变量未赋值
		+ 变量被unset()

14. 运算符
	+ 连接运算符
		+ .运算符：将两个字符串连接起来
		+ .=运算符：将一个字符串连接到另一个字符串的末尾：` $a .= $b;`
	+ 算术运算符
		+ 加减乘除取模：+、-、*、/、%
	+ 比较运算符
		+ 等于：==
		+ 不等于：!=
		+ 大于：>
		+ 小于：<
		+ 大于等于：>=
		+ 小于等于：<=
	+ 逻辑运算符
		+ 与：&&
	+ 三目运算符
		+ 语法：` (条件) ? 值1 : 值2;`
		+ 示例：` $a = (10 > 5) ? "大于" : "小于";`
	+ 联合运算符
	```
	$a = [
		"a" => "ab",
		"b" => "bb",
	];
	$b = [
		"c" => "cc",
		"d" => "dd",
	];
	$c = $a + $b;
	var_dump($c);	//Array ( [a] => ab [b] => bb [c] => cc [d] => dd )
	```

# 三、流程控制
1. if语句
	+ ` if($x)  //已知$x为整型，当$x不等于0时，执行if语句块，否则执行else语句块`

2. foreach语句
	+ 语法：
		```
		foreach($array as $value) {
			// 代码块
		}
		```
	+ 示例：
		```
		$a = [1, 2, 3, 4, 5];
		foreach($a as $value) {
			echo $value;
		}
		```

3. 中断语句
	+ break：结束当前循环
	+ continue：结束当前循环，继续下一次循环
	+ exit：结束当前脚本
	+ return：结束当前函数
	+ die：结束当前脚本

# 五、超全局变量
1. 定义
	+ 超全局变量是在所有作用域中可用的内置变量
	+ 在函数或方法中无需执行global $variable;就可以访问它们
	+ 超全局变量列表：
		+ $GLOBALS：包含所有全局变量的全局组合数组
		+ $_SERVER：包含所有服务器变量
		+ $_GET：包含所有GET变量
		+ $_POST：包含所有POST变量
		+ $_FILES：包含所有文件上传变量
		+ $_COOKIE：包含所有Cookie变量
		+ $_SESSION：包含所有Session变量
		+ $_REQUEST：包含所有请求变量
