# 二、php基本语法
2. php文件格式
	+ 代码需要在尖括号里面：
		```
		<?php
			// 这是一个注释
			echo "hello world";
		?>
		```
	+ <?php ?>

2. 注释
	+ 单行注释：
		+ `// 这是一个注释`
		+ ` # 这是一个注释`
	+ 多行注释：`/* 这是一个注释 */`

3. 拼接字符串
	+ 可以使用`.`运算符来拼接字符串
	+ 例如：`$a = "hello"; $b = "world"; echo $a . $b;`

4. 单引号和双引号
	+ 单引号：`'hello world'`
		+ 不可以嵌入变量
	+ 双引号：`"hello world"`
		+ 可以嵌入变量
	```
		$str = "what the fuck";
		echo "output $str";
		echo "<br>";
		echo 'output $str';
	```

5. 调试，查看变量的值
	+ var_dump()函数：
		+ 可以查看变量的类型和值
		+ 没有返回值
		+ 例如：`var_dump($a);`
		+ 输出结果：`int(10)`

6. 变量
	+ 以`$`开头，跟着变量的名称，用=进行赋值，将右边的值赋给变量。
	+ 变量名只能包含字母、数字和下划线
	+ 变量名不能以数字开头

7. 变量作用域
	+ 我们在函数内访问全局变量，需要使用`global`关键字引入
	+ static关键字：
		+ 可以在函数内定义一个静态变量，静态变量的值在函数调用之间保持不变
	+ 例如：
	```
		//函数内使用全局变量
		function test(){
			global $a;
			echo $a;
		}
		$a = 10;
		test();

		//static关键字
		function test2(){
			static $a = 0;
			$a++;
			echo $a;
		}
		test2();	//1
		test2();	//2
		test2();	//3
	```

8. 常量
	+ 常量名要用大写字母
	+ 语法：
		`define("常量名", 值);    //已弃用`
		`const 常量名 = 值;`
	+ 例如：
		```
			define("PI", 3.1415926);	//已弃用
			echo PI;	//3.1415926

			const PI = 3.1415926;
			echo PI;	//3.1415926
		```

9. 变量
	+ 变量类型：整型、浮点型、字符串、布尔型、数组、对象、资源、空类型
	+ 类型检测：
		+ is_int()：检测变量是否为整型
		+ is_float()：检测变量是否为浮点型
		+ is_string()：检测变量是否为字符串
		+ is_bool()：检测变量是否为布尔型
		+ is_array()：检测变量是否为数组
		+ is_object()：检测变量是否为对象
		+ is_resource()：检测变量是否为资源
		+ is_null()：检测变量是否为空
	+ 类型转换：
		+ intval()：将变量转换为整型
		+ floatval()：将变量转换为浮点型
		+ strval()：将变量转换为字符串;NULL和布尔型会转换为空字符串
		+ boolval()：将变量转换为布尔型
		+ arrayval()：将变量转换为数组
		+ objectval()：将变量转换为对象
		+ resourceval()：将变量转换为资源
		+ nullval()：将变量转换为空

10. 浮点数
	+ 浮点型数据，是存在精度问题的
	+ 永远不要相信浮点数结果精确到了最后一位，也永远不要比较两个浮点数是否相等。

11. 字符串
	+ 双引号和单引号的区别：
		+ 双引号可以解析变量，单引号不能解析变量
		+ 双引号可以解析转义字符，单引号不能解析转义字符
	+ 字符串定义类型：单引号形式、双引号形式、heredoc形式
	+ 字符串转换
		+ strval()：将变量转换为字符串;NULL和布尔型会转换为空字符串
	+ 字符串长度：strlen()函数
		+ 例如：`echo strlen("hello");`
		+ 输出结果：5
	+ 字符串拼接：.运算符
		+ 例如：`echo "hello" . "world";`
		+ 输出结果：helloworld
	+ 字符串截取：substr()函数
		+ 例如：`echo substr("hello", 1, 3);`
		+ 输出结果：ell
	+ 字符串替换：str_replace()函数
		+ 例如：`echo str_replace("hello", "world", "hello world");`
		+ 输出结果：world world

12. 数组
	+ 数组类型：索引数组、关联数组
	+ 数组定义：
		+ 索引数组：`$arr = array(1, 2, 3);`
		+ 关联数组：`$arr = array("name" => "John", "age" => 30);`
	+ 数组操作：
		+ 添加元素：`$arr[] = 4;`
		+ 删除元素：`unset($arr[0]);`
		+ 修改元素：`$arr[0] = 100;`
