# 一、PHP简历
1. 定义
	+ PHP是一种服务端的HTML脚本语言，是一种简单的、面向对象的、解释型的脚本语言。
	+ 改语言让web开发人员快速地书写动态生成的网页

2. php文件格式
	+ 代码需要在尖括号里面：
		```
		<?php
			// 这是一个注释
			echo "hello world";
		?>
		```
	+ <?php ?>


# 二、php基本语法
2. php文件格式
	+ 代码需要在尖括号里面：
		```
		<?php
			// 这是一个注释
			echo "hello world";
		?>
		```
	+ <?php ?>

2. 注释
	+ 单行注释：
		+ `// 这是一个注释`
		+ ` # 这是一个注释`
	+ 多行注释：`/* 这是一个注释 */`

3. 拼接字符串
	+ 可以使用`.`运算符来拼接字符串
	+ 例如：`$a = "hello"; $b = "world"; echo $a . $b;`

4. 单引号和双引号
	+ 单引号：`'hello world'`
		+ 不可以嵌入变量
	+ 双引号：`"hello world"`
		+ 可以嵌入变量
	```
		$str = "what the fuck";
		echo "output $str";
		echo "<br>";
		echo 'output $str';
	```

5. 调试，查看变量的值
	+ var_dump()函数：
		+ 可以查看变量的类型和值
		+ 没有返回值
		+ 例如：`var_dump($a);`
		+ 输出结果：`int(10)`

6. 变量
	+ 以`$`开头，跟着变量的名称，用=进行赋值，将右边的值赋给变量。
	+ 变量名只能包含字母、数字和下划线
	+ 变量名不能以数字开头

7. 变量作用域
	+ 我们在函数内访问全局变量，需要使用`global`关键字引入
	+ static关键字：
		+ 可以在函数内定义一个静态变量，静态变量的值在函数调用之间保持不变
	+ 例如：
	```
		//函数内使用全局变量
		function test(){
			global $a;
			echo $a;
		}
		$a = 10;
		test();

		//static关键字
		function test2(){
			static $a = 0;
			$a++;
			echo $a;
		}
		test2();	//1
		test2();	//2
		test2();	//3
	```

8. 常量
	+ 常量名要用大写字母
	+ 语法：
		`define("常量名", 值);    //已弃用`
		`const 常量名 = 值;`
	+ 例如：
		```
			define("PI", 3.1415926);	//已弃用
			echo PI;	//3.1415926

			const PI = 3.1415926;
			echo PI;	//3.1415926
		```
