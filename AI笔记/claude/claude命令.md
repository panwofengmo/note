1. rewind命令
	+ 作用：回滚代码
	+ 用法：输入 /rewind，会弹出对话历史中各个修改文件的检查点（checkpoint）
	+ 选择一个检查点后，对话和代码文件都会恢复到该时间点的状态
	+ 适用场景：当 Claude Code 改坏了代码、方向偏离预期时，快速撤销到之前的状态

2. resume命令
	+ 作用：恢复之前的对话会话
	+ 用法：输入 /resume，会列出历史对话记录
	+ 选择一个对话后，可以继续之前的上下文和工作进度
	+ 适用场景：退出 Claude Code 后想接着上次未完成的任务继续工作

3. init命令
	+ 作用：为项目初始化 CLAUDE.md 文件
	+ 用法：在项目根目录输入 /init，Claude Code 会分析代码库结构并生成 CLAUDE.md
	+ CLAUDE.md 包含项目架构、编码规范、常用命令等信息，供后续对话中 Claude 自动参考
	+ 适用场景：新项目启动时，或现有项目缺少 CLAUDE.md 需要补全时

4. context命令
	+ 作用：查看当前对话的上下文使用情况
	+ 用法：输入 /context，显示上下文窗口的占用比例、各部分大小等信息
	+ 适用场景：对话较长时，查看上下文是否即将耗尽，决定是否需要压缩或开启新会话
