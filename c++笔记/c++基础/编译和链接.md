1. 分离式编译
	+ 头文件(.h)在另一个文件夹，包含函数声明；
	+ 实现文件和调用main函数在另一个文件夹，在main函数文件include头文件可实现编译
	+ ` g++ fact.cpp factMain.cpp`


2. 编译和链接
	+ 编译会将每一个cpp文件编译成对应的obj文件
	+ 而链接则是将这些obj文件链接成一个exe文件
	+ 举例：
		+ main.cpp调用log.cpp的log函数，那么main.cpp的log函数声明会让编译器相信有个log函数
		+ 而链接则是将log.obj文件中，拿出log的定义，放到二进制文件中

3. include的作用
	+ 指定想要包含的文件，预处理器打开那个文件，阅读所有内容，然后粘贴到你写的文件中
	```
	//endBrace.h
	}

	//math.cpp
	int multiply(int a, b)
	{
		int ret = a * b;
		#include endBrace.h;
	```

4. static函数
	+ 静态函数，则此函数只会在这个cpp文件被调用，不会被其他地方调用
	+ 如果其他地方调用了这个函数，会报错(除非自己有个相同的static函数)
	+ 如果static有个无定义的函数，则不会发生链接错误
 
5. 头文件
	+ C++中，任何以#开头的东西，被称为预处理器命令或预处理器指令。在实际编译此文件之前，就被先处理
