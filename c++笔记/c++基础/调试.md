# 一、 linux + vscode + c/c++插件
1. vscode下载c/c++插件
2. 安装gdb: ` sudo apt install gdb`
3. 点击左边的运行和调试，然后创建launch.json文件，选择cmake
4. 创建一个cpp文件，然后点击右下角的添加配置
5. 选择(gdb) 启动
6. 修改"program"选项，这里是指定的执行文件
	```
	"program": "${workspaceFolder}/a.out",
	```
7. 如果需要打断点，那么编译的时候需要加上-g选项
	```
	g++ -g hello.cpp
	```
8. 启动` (gdb) 启动`
