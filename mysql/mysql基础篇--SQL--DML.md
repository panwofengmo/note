+ DML(数据操作语言, 用来对数据库中的数据进行增删改)
+ DQL(数据查询语言, 用于查询数据库中表的数据)

# 一、插入数据--INSERT
1. 给指定字段添加数据
	+ 语法：`INSERT INTO 表名 (字段1, 字段2, ...) VALUES (值1, 值2, ...);`
	+ 注意：
		+ 字段和值的顺序要一一对应
		+ 字符串和日期类型需要用单引号括起来
		+ 插入的数据大小要在字段的长度范围内

2. 给所有字段添加数据
	+ 语法：`INSERT INTO 表名 VALUES (值1, 值2, ...);`

3. 批量添加数据
	+ 语法1：`INSERT INTO 表名 (字段1, 字段2, ...) VALUES (值1, 值2, ...), (值1, 值2, ...), ...;`
	+ 语法2：`INSERT INTO 表名 VALUES (值1, 值2, ...), (值1, 值2, ...), ...;`


# 二、更新数据--UPDATE
1. 修改单条数据
	+ 语法：`UPDATE 表名 SET 字段1=值1, 字段2=值2, ... [WHERE 条件];`
	+ 注意：
		+ 条件可以省略, 但建议加上, 否则会修改所有数据
	+ 示例：
	+ ` UPDATE employee SET name = 'itheima' WHERE id = 1;`	// 修改id为1的员工姓名为itheima


# 三、删除数据--DELETE
1. 语法：`DELETE FROM 表名 [WHERE 条件];`
	+ 注意：
		+ 条件可以省略, 但建议加上, 否则会删除所有数据
	+ 示例：
	+ `DELETE FROM employee WHERE id = 1;`	// 删除id为1的员工
