1. 运行go文件
	+ 只运行，不生成exe文件：` go run helloworld.go`

2. 编译
	+ 生成可执行程序：
	```
		go build helloworld.go	//生成可执行文件: helloworld
		./helloworld
	```
	+ ` go build helloworld.go`

3. go语言的组成
	+ Go代码是使用包来组织的，包类似其他语言中的库和模块
	+ 一个包由一个或多个.go源文件组成，放在一个文件夹中，该文件夹的名字描述了包的作用
	+ 每一个源文件的开始都用package声明，main.go里面是package main，指明了这个文件属于哪个包
	+ 名为main的包比较特殊，它用来定义一个独立的可执行程序，而不是库
	+ 不管在什么程序中，main做什么事情，它总是程序开始执行的地方

4. import
	+ 作用：用来导入包
	+ import声明必须跟在package声明之后

5. 函数
	+ 声明：func关键字 + 函数名 + 参数列表 + 返回只列表
