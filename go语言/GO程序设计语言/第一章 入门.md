1. 运行go文件
	+ 只运行，不生成exe文件：` go run helloworld.go`

2. 编译
	+ 生成可执行程序：
	```
		go build helloworld.go	//生成可执行文件: helloworld
		./helloworld
	```
	+ ` go build helloworld.go`

3. go语言的组成
	+ Go代码是使用包来组织的，包类似其他语言中的库和模块
	+ 一个包由一个或多个.go源文件组成，放在一个文件夹中，该文件夹的名字描述了包的作用
	+ 每一个源文件的开始都用package声明，main.go里面是package main，指明了这个文件属于哪个包
	+ 名为main的包比较特殊，它用来定义一个独立的可执行程序，而不是库
	+ 不管在什么程序中，main做什么事情，它总是程序开始执行的地方

4. import
	+ 作用：用来导入包
	+ import声明必须跟在package声明之后

5. 函数
	+ 声明：func关键字 + 函数名 + 参数列表 + 返回只列表

6. 规范
	+ 函数func关键字必须和“{”在同一行
	+ 换行符可以在“+”后面，但不能在前面

7. 隐式初始化
	+ 如果变量没有明确地初始化，它将隐式地初始化为这个类型的空值
	+ string变量：空字符串""
	+ int变量：0

8. for循环语句
	+ 类型1：` for init; cond; post{...}`
	```
		for i = 1; i < 10; ++i{
			fmt.Print(i)
		}
	```
	+ 类型2：`for index, value := range collection {...}` （遍历数组、切片、map、字符串等）
	```
		for i, v := range arr {
			fmt.Println(i, v)
		}
	```

9. 空标识符` _`
	+ 空标识符可以用在任何语法需要变量名，但是程序逻辑不需要的地方

10. 声明
	+ 字符串的声明
	```
		s := ""
		var s string
		var s = ""
		var s string = ""
	```
