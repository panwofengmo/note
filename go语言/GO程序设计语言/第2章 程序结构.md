# 一、名称
## 1. 有效性
+ 如果变量声明在函数外，它将对包里面的所有源文件可见
+ 实体的第一个字母的大小写决定了其可见性：
	+ 大写字母开头的变量或函数是导出的，可以在包外部访问。如: ` fmt.Println()`
	+ 小写字母开头的变量或函数是未导出的，只能在包内部访问
+ 通常，名称的作用域越大，就使用越长且更有意义的名称


# 二、声明
+ 有4个主要的声明：变量(var)、常量(const)、类型(type)、函数(func)
+ go文件结构
```
//每一个文件以package声明开头，表名文件属于哪个包
package main

//导入声明
import "fmt"

//包级别的类型、常量、变量、函数、结构体等声明，不区分顺序
const boilingF = 212.0

func main(){
	var f = boilingF
	var c = (f - 32) * 5 / 9
	fmt.Printf("boiling point = %gF or %g C", f, c)
}
```
+ ` %g`：输出浮点数，保留小数点后6位。会自动去除尾部无意义的零
